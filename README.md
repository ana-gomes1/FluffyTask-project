<h1>TaskyFluffy - Gerenciador de tarefas</h1>

o projeto TaskyFLuffy é um website que simula uma aplicação com o objetivo de ser um gerenciador de tarefas, com funções de criar, excluir e editar as tarefas como concluída ou incompleta.


![Design sem nome](https://github.com/user-attachments/assets/dc83228d-8eb2-4c76-9059-3f0baf411e0e)


<h1>Funções</h1>

-  <b>Usabilidade e experiência do usuário</b>: Focado em um design intuitivo e organizado (UI/UX design)
-  <b>Navegabilidade</b>: Exibição de telas dinâmica utilizando JavaScript
-  <b>Gerenciar tarefas</b>: Sistema com gerenciamento de tarefas

<h1>Linguagens e Ferramentas utilizadas</h1>

<h3><samp>Front-end</samp></h3>

- <b>HTML</b>: Estruturação dos elementos da aplicação
- <b>TailwindCSS</b>: Estilização e responsividade dos conteúdos
- <b>JavaScript</b>: Interatividade das páginas e validação das tarefas

<h3><samp>Back-end</samp></h3>

- <b>PHP</b>: Logica do cadastro, alteração e exclusão das tarefas
- <b>JSON</b>: Armazenamento das tarefas em formato JSON

<h1>Como Rodar o Projeto</h1>

<h3>Pré-Requisitos</h3>

- Ambiente de Desenvolvimento PHP instalado (Exemplo: XAMPP, LAMP, etc)

<h3>Passo a Passo</h3>

<b>1. Clone o repositório</b><br>

```
git clone https://github.com/ana-gomes1/TaskyFluffy-projeto.git
```
<br>
<b>2. Mova a pasta do projeto para o diretório do servidor</b><br><br>

- No XAMPP
```
C:\xampp\htdocs
```

- No LAMP
```
/var/www/html
```
<br>
<b>3. Inicie o servidor Apache</b><br><br>

- Abra a aplicação do ambiente de desenvolvimento (XAMPP, LAMP, etc) e inicie ambos os servidores

Obs.: O servidor Apache por padrão roda na porta 80.
<br><br>

<b>4. Acesse o localhost</b>

- Vá para o navegador e digite a seguinte URL para utilizar o projeto no servidor web:

```
http://localhost/
```
